## Project Overview  
**SOC-cessful Schedule** simplifies scheduling and booking by providing an interactive dashboard for managing meetings, polls, and alternative requests. The design reflects the identity of McGill's **School of Computer Science (SOCS)** while maintaining modern aesthetics.

Some sample features are:
- üîí **User Authentication**: Secure login and registration.  
- üóÇ **Member Dashboard**: Central hub for managing:  
   - üìú **Booking History**: View past bookings.  
   - ‚öôÔ∏è **Preferences**: Update personal settings.  
   - üóì **Meeting Management**: Create and organize meetings.  
   - üìä **Polls**: Gather user inputs through polls.  
- üìÜ **Booking System**: Share links for others to book time slots.  
- üîÑ **Alternate Time Requests**: Propose or modify meeting schedules.
